# Project Name

Built with Next.js (App Router) + Sanity CMS.

## Setup

```bash
cp .env.local.example .env.local
# fill in NEXT_PUBLIC_SANITY_PROJECT_ID from sanity.io/manage
pnpm install
pnpm dev
```

## Commands

```bash
pnpm dev    # localhost:3000
pnpm build
pnpm lint
```

Sanity Studio is available at `/studio`.
